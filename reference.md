## Glossary

~~~
{:auto_ids}

cell
:	an individual piece of data in a spreadsheet, value for one variable related to one observation. Indexed by the column and row number (ex. 'A1' is the first cell).

column
:	vertical series of cells in a spreadsheet, commonly holds data for one variable. Indexed using letters (ex. 'A' is the first column).

row
:	horizontal series of cells in a spreadsheet, commonly holds data for one observation. Indexed using numbers (ex. '1' is the first row).

~~~
